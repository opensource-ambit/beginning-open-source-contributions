Open source in brief:
=============
* Programs(code) used to build and run the software are freely available, which users may view, modify, adopt, and share.
* License: generally included as plain text description, mentions original creators, and certain rules.
* Rules, common ones: When you share modified code, include the original license text description, or modified versions also be open source.
* Software built through (usually, global) collaboration.
* Contributors: review, test, improve the code and documentation, often using version control systems like Git to manage changes.

By contributing to open source, one can improve documentation, communication, version control, and technical skills.

In context of this `README.rst <README.rst>`_ file, contributing means, support through documentation or code, and not financial, donate, contributions.
